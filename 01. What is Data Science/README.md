# What is Data Science?

## Summary
An introductory course on the core concepts, history, and real-world applications of data science. Learners explore the role of data scientists, emerging technologies like machine learning, and the growing demand for data-driven decision-making across industries.

## Main Topics

* Definition, scope, and evolution of data science

* Applications in business and society

* Career paths and roles within the data science field

* Machine learning and deep learning fundamentals

## Key Skills Learned

* Explain what data science is and why it matters today

* Identify diverse pathways into a data science career

* Understand how organizations use data science for insight and prediction

* Summarize professional advice from experienced data scientists
