# Data Science Methodology

## Summary
Learn structured approaches to solving data science problems using methodologies like CRISP-DM. Develop skills in problem definition, data analysis, modeling, and communicating insights.

## Main Topics

* Importance of data science methodology and CRISP-DM framework

* Analytic model types: predictive, descriptive, classification

* Data preparation, modeling, and deployment

## Key Skills Learned

* Apply structured methodology to real-world data problems

* Select and evaluate appropriate analytic models

* Analyze, model, and communicate insights effectively
